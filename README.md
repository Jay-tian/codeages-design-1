# Codeages Design

a Web Design Language

### install

```
yarn add codeages-design
```

### Quick Start

```
import 'jquery';
import cd from 'codeages-design';
import 'codeages-design/src/less/codeages-design.less';

window.cd = cd;

// example
cd.loading();
```

### Browser Support

Modern browsers and Internet Explorer 9+
